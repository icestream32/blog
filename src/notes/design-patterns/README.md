这个系列是我在读程杰老师的《大话设计模式》的一些读书笔记，主要是为了加深对设计模式的理解，同时也是为了记录一些自己的思考和总结。

文章里主要对书本中的示例代码用TypeScript和Golang进行了实现，同时也对书本中的一些概念进行了一些解释和总结。
