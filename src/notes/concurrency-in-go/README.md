最近公司领导突然向我发问：“对Go语言感兴趣不？一些网络通信的组件可能会用到Go语言。”身为一个初入职场的毕业生小白，那必然得回应：“十分滴感兴趣（虽然的确是感兴趣）。”因此，为了抓住这个机会，本就在学习Go语言的我直接进入到了Go并发编程的学习之中。

有了这个念头就开始了Go并发编程的学习，先前的Go学习都是在[极客时间](https://time.geekbang.org/)学习的，因此同样也了解到了在Go并发编程这方面有一位大牛——晁岳攀（@鸟窝）老师，他对Go并发编程有着深入的理解，因此我也选择了他的《深入理解Go并发编程》这本书作为我的学习资料。

这个系列是我在读晁岳攀老师的《深入理解Go并发编程》的一些读书笔记，主要是为了加深对Go并发编程的理解，同时也是为了记录一些自己的思考和总结。
