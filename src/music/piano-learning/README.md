编程是主业，音乐是生活！工作之余学习一门乐器，有助于开拓视野，提高专注力，减轻压力！因此我盯上了钢琴这个乐器，希望通过学习钢琴，提高自己的音乐素养，同时也希望通过这个系列文章，记录自己的学习过程，分享自己的心得体会。

但是有一说一，虽然说是钢琴学习，但是我买的是电子琴（买不起钢琴，悲）。不过小问题，我的电子琴是61键，钢琴是88键，所以在学习的过程中，可能会有一些区别，但是基本的原理是一样的。
