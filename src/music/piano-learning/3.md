这篇博客对应视频第三集，视频中主要介绍了和弦和分解和弦的概念，以及如何在钢琴上演奏三和弦和分解三和弦。

## 和弦

和弦是音乐中的基本元素，是由三个或三个以上音符同时演奏而成的音乐段落。在刚入门阶段我们主要学习的是三和弦。

认识一下三和弦的分类：

- 主和弦：对应到字母是C E G

- 属和弦：对应到字母是G E C

## 分解和弦

分解和弦是将和弦中的音符分开演奏，这样可以更好地理解和弦的构成。

在钢琴上演奏三和弦时，我们可以将和弦中的音符分开演奏，这样就形成了分解和弦。

## 转位

转位是指将和弦中的音符的位置进行调换，这样可以使和弦的音色更加丰富。

## 练习

1. 左右手练习D大调，D大调的排列如下：

D E F# G A B C# D

2. 三和弦练习

按照下面的图示，逐一用手指对应上去，一开始可以先两根手指同时按下，之后熟练后再切换到三根手指，听钢琴发出的声音，一定要让其同时发声。

转位后的和弦位置如下（拇指到小指的数字分别为1到5，中文的一二三表示按下的顺序）：

> 左手

```
            C  D  E  F  G  A  B  C  D  E  F

            ^  ^  ^  ^  ^  ^  ^  ^  ^  ^  ^
            |  |  |  |  |  |  |  |  |  |  |
            |  |  |  |  |  |  |  |  |  |  |
            |  |  |  |  |  |  |  |  |  |  |

一：        5     3     1
二：              5     4        1
三：                    5        3     1
```



> 右手

```
            C  D  E  F  G  A  B  C  D  E  F

            ^  ^  ^  ^  ^  ^  ^  ^  ^  ^  ^
            |  |  |  |  |  |  |  |  |  |  |
            |  |  |  |  |  |  |  |  |  |  |
            |  |  |  |  |  |  |  |  |  |  |

一：        1     3     5
二：              1     2        5
三：                    1        3     5
```

左右手练习逐步熟悉每个转位的和弦位置，让肌肉记忆帮助我们更好地演奏和弦。

3. 分解和弦练习

运用上手腕的力量，将上述和弦的音符分开演奏，从左到右，从右到左，练习顺序如下：

> 左手

```
5 ------> C （表示小指按下C，之后同理）
3 ------> E
1 ------> G

动作：在拇指按下G后将小指挪到E并按下，之后再由无名指按下G。

5 ------> E
4 ------> G
1 ------> C

动作：在拇指按下C后将小指挪到G并按下，之后再由中指按下C。

5 ------> G
3 ------> C
1 ------> E

动作：在拇指按下E后将小指挪到C并按下，之后再由中指按下E。

5 ------> C
3 ------> E
1 ------> G

动作：此时开始折返

1 ------> G
3 ------> E
5 ------> C

动作：在小指按下C后将拇指挪到E并按下，之后再由中指按下C。

1 ------> E
3 ------> C
5 ------> G

动作：在小指按下G后将拇指挪到C并按下，之后再由无名指按下G。

1 ------> C
4 ------> G
5 ------> E

动作：在小指按下E后将拇指挪到G并按下，之后再由中指按下C。

1 ------> G
3 ------> E
5 ------> C
```

> 右手

右手同理左手，这里就不放详细过程了。

```
1 ------> C
3 ------> E
5 ------> G

1 ------> E
2 ------> G
5 ------> C

1 ------> G
3 ------> C
5 ------> E

1 ------> C
3 ------> E
5 ------> G

5 ------> G
3 ------> E
1 ------> C

5 ------> E
3 ------> C
1 ------> G

5 ------> C
4 ------> G
1 ------> E

5 ------> G
3 ------> E
1 ------> C
```

一开始手指一一对应可能费点时间，但是手指熟练形成肌肉记忆后就很舒服了。
