## 构成

首先先上图，这个是完整五线谱：

![五线谱](https://images.icestream32.cn/images/2025/02/23/82c6f422f53ac26fc8aef84d136a25dd.png)

接下来分别介绍五线谱的各个部分：

### 大括弧

五线谱最左侧的大括号即为大括弧。如果大括弧将两段五线谱括起来的话，表示需要同时演奏上下部的音符，这种情况一般出现在钢琴谱中，此时左手演奏下部的音符，右手演奏上部的音符。

### 谱号

谱号位于五线谱的最左侧，用来表示音符的音高，我们现在要学习的是两种谱号：高音谱号和低音谱号。

- ![高音谱号](https://upload.wikimedia.org/wikipedia/commons/f/ff/GClef.svg)：高音谱号，又叫G谱号，表示后面的五线谱是高音区，对应右手。
- ![低音谱号](https://upload.wikimedia.org/wikipedia/commons/c/c5/FClef.svg)：低音谱号，又叫F谱号，表示后面的五线谱是低音区，对应左手。

### 小节线

即图中一段段竖线，用来将五线谱分割成小节，方便实现音符的节奏。

### 五线谱

五线谱由五条平行的线组成，针对钢琴谱上下部的五线，对应到钢琴的琴键分别如下：

1. **上部五线**

```
-------------------- F
E
-------------------- D
C
-------------------- B
A
-------------------- G
F
-------------------- E
```

便携记忆方法：

- 空白部分：脸（FACE），face to space（空白即脸）
- 线部分：恶（e）搞（gao）贝（bei）多（duo）芬（fen）

2. **下部五线**

```
-------------------- A
G
-------------------- F
E
-------------------- D
C
-------------------- B
A
-------------------- G
```

便携记忆方法：

- 空白部分：爱（ai）吃（chi）鹅（e）肝（gan）
- 线部分：隔（ge）壁（bi）的（de）父（fu）爱（ai）

3. **中间部分**

中间部分即为中央C，对应到钢琴上的中央C，如下：

```
-------------------- E 上半

中间是中央C

-------------------- A 下半
```

有一说一中央C和E之间的D、C和A之间的B去哪了，以后会学到钢琴谱中间部分还有一些符号，那些符号对应的就是这两个音符。

4. **拓展线**

有时候演奏的音高会超过五线谱的范围，这个时候就需要拓展线谱，即在五线谱上方或下方加上一条或多条线，这些线称为拓展线。

我们需要记忆的就是常用的上下三条拓展线，如下：

- 上三条拓展线及其间隔音符：

```
F
-------------------- E
D
-------------------- C
B
-------------------- A
G
```

- 下三条拓展线：

```
F
-------------------- E
D
-------------------- C
B
-------------------- A
G
```

- 便携记忆方法：ACE （CF 的 MVP显示），然后上下拓展线中中间的都是C。

> 以上抽象的记忆方法均来自视频里的内容，既视感太强烈了看一遍就记住了哈哈哈哈哈哈哈
