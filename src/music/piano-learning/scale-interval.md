## 音阶

音阶是音乐中的基本元素，是一组音符按照一定的规律排列而成的音乐段落。

音阶的种类有很多，常见的有：

- 大调音阶
- 小调音阶

我们现在主要要学的都是大调音阶。

视频中老师给我们演示了C大调和G大调音阶的演奏方法

C大调音阶的音符是：

```
C D E F G A B C
```

G大调音阶的音符是：

```
G A B C D E F# G
```

这里的#代表升号，也就是说F#是F升一个半音得到的音符。（半音的概念会在下面音程中讲到）
同理还有降号，降号的符号是b，比如Fb就是F降一个半音得到的音符。

## 音程

音程是两个音符之间的距离，以钢琴上的黑白键举例：

- 两个相邻的琴键之间的音程叫做半音，比如说C和C#之间的音程
- 两个相隔一个琴键的音程叫做全音，比如说C和D之间的音程

音程有很多种，常见的有：

- 八度音程，比如C和C之间的音程
- 大三度音程，比如C和E之间的音程
- 小三度音程，比如C和Eb之间的音程

::: important
在之后的联系中，我们必须以音程的形式来练习记忆音符，通过这样练习可以使得肌肉记忆更加深刻，看谱子的时候就可以更快的找到对应的琴键。
