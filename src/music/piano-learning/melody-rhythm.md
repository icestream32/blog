## 休止符

休止符适用于音乐的乐谱上，标记音乐暂时停顿和停顿时间长短的记号。休止符的使用，可制造出音乐乐句中不同的情绪表达。

休止符的种类如下：

## 节拍强弱

常见的不同节拍的强(Strong)弱(Weak)程度如下：

| 节拍 | 强弱程度 |
| :---: | :---: |
| $\frac{2}{4}$ | 强弱中弱 |
| $\frac{3}{4}$ | 强弱弱 |
| $\frac{2}{4}$ | 强弱 |
| $\frac{6}{8}$ | 强弱弱强弱弱 |

::: tip
这里的强弱指的是先前学习的[音乐区间理论](/music/piano-learning/interval-theory.md#其他的音乐符号)强弱音，忘了可以回去复习一下。
:::

## 休止符作用下的旋律

休止符作用下的旋律，会遵循一定的原则，理解这些原则有助于我们在看谱子和作曲的时候更好地理解旋律。

- 强拍前置原则

按照从左到右的顺序，休止符下的弱拍一定不能出现在强拍之前。

- 弱拍后置原则

按照从左到右的顺序，强弱拍上可以放一个休止符表示这两个拍子停顿。

这些原则本质上是在维护**节拍张力层级**，讲得挺抽象的，实际上如果不按照这些原则作曲演奏的话，那么整个音乐演奏起来会非常奇怪，主次不分明。

强拍如同建筑承重墙，不可被休止符掏空；弱拍则是可调节的隔断，但不可反向破坏结构骨架。
