## 硬件选择

## 装机过程

因为本人有过装台式电脑的经验，因此这个旧时代的主机装机还挺简单的，要看说明书的地方还得是NAS机箱开机线与主板连接的位置（得照着主板说明书一个个对，不然开机都开不了）。

其次就是这个NAS机箱和机械硬盘的固定方式了，乔思伯他们家的机箱比较特殊，是通过胶套和机箱里的卡槽固定的，因此需要先用螺丝固定好胶套，再把硬盘直接推进去即可。

::: tip
在装好CPU、主板、内存条、散热器后，可以使用电源测试一下主板是否正常工作，如果主板指示灯只有VGA灯亮着（说明主板没有连接显示器），那么主板就没有问题。
:::

## 软件选择

操作系统选择[飞牛OS](https://www.fnnas.com/)，这个系统是基于Debian的，社区比较活跃，有比较多的教程可以参考，相比于黑群晖来说，不容易被制裁（）。

### 创建镜像

在飞牛OS页面下载镜像后，使用开源的镜像制作工具[rufus](https://rufus.ie/)，选择飞牛镜像文件，选择U盘，点击开始即可，写入成功后，U盘就变成了启动盘。

> 注意：写入前rufus可能会提示警告，忽略继续执行即可。

### 安装系统

将U盘插入NAS机子，开机，按DEL进入BIOS，选择U盘启动，进入飞牛OS的安装界面，选择系统安装在固态硬盘等待安装完成，重启电脑后，此时终端界面会显示飞牛OS的登录Web IP，在电脑浏览器输入IP地址加端口即可访问飞牛OS操作界面，并进行初始化设置。

初始化界面需要做的操作如下：

- 设置NAS的名称
- 设置管理员密码
- 在系统设置中开启SSH远程连接
- 下载安装飞牛内置的虚拟机

::: tip
这里网线的连接一定要是以路由器为DHCP服务器，NAS为DHCP客户端，理由：

飞牛OS的终端一开始初始化的时候是进不了的，也就是说无法通过输入用户名密码登录终端界面，只有访问终端界面提供的IP地址，设置好管理员用户密码后，才能通过键盘显示器登录终端。
:::
